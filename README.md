# Pizzeria
Fun little project I've done to learn Vue.  
Demo version without backend hosted [here](https://v1laz.github.io/PizzeriaWeb/#/)

## Project setup
```
npm install
```

## Run backend
```
npm run backend
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
